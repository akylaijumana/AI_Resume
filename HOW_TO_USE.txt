================================================
   AI RESUME GENERATOR - SIMPLE SETUP GUIDE
================================================

STEP 1: GET YOUR OPENAI API KEY
--------------------------------
1. Go to: https://platform.openai.com/
2. Sign up or log in
3. Click on "API Keys" in the left menu
4. Click "Create new secret key"
5. Copy the key (it starts with "sk-")

STEP 2: SET UP YOUR API KEY
----------------------------
1. In this folder, create a new file called ".env" (no name, just .env)
2. Open the .env file in Notepad
3. Type this line (replace with your actual key):
   OPENAI_API_KEY=sk-your-actual-key-here
4. Save and close

STEP 3: RUN THE PROGRAM
------------------------
Option A - Using the run script (EASIEST):
   - Double-click on "run.ps1"
   - If Windows asks, choose "Run with PowerShell"

Option B - Using PowerShell:
   - Right-click in this folder
   - Choose "Open in Terminal" or "Open PowerShell here"
   - Type: .\run.ps1
   - Press Enter

Option C - Manual way:
   - Open PowerShell in this folder
   - Type: .\.venv\Scripts\Activate.ps1
   - Then type: python resume_generator.py

STEP 4: USE THE PROGRAM
------------------------
1. Fill in your information in the form
2. Click "Generate Resume"
3. Wait a few seconds
4. Your AI-generated resume will appear
5. Click "Save as PDF" to save it

IMPORTANT NOTES:
----------------
- You need internet connection to use this
- OpenAI charges very little (about $0.001 per resume)
- You need to add at least $5 credit to your OpenAI account
- The GUI is simple and clean - just fill and click!

TROUBLESHOOTING:
----------------
If you get an error about API key:
   - Make sure you created the .env file correctly
   - Check that your API key is correct
   - Make sure there are no extra spaces

If PowerShell won't run the script:
   - Try: Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
   - Then try running again

Need help? Check README.md for more details.
